# NEXTWAY School Management System — Frontend

## Quick Start
```bash
npm install
npm run dev        # http://localhost:5173
```

## Environment
Create `.env` (already included):
```
VITE_API_URL=http://localhost:5000/api/v1
```

## Demo Login (no backend needed)
Click **"Enter Demo"** on the login screen to use mock data.

## With Backend
Set `VITE_API_URL` to your running API URL, then login with real credentials from seed.
